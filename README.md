# Restaurant-app-
this app for ordering food for a new established restaurant 
Features:
 List of menu items displayed using RecyclerView
 Click on an item to open detail screen (Intent passes name, price, description, image)
 Static menu loaded locally inside MainActivity
 
Client (Android):Kotlin
XML UI Layouts (not Compose)
RecyclerView + ViewBinding
Architecture: No MVVM yet (simple Activity + Adapter)
Storage: None (menu items are hardcoded)
Navigation: Classic Intent–based navigation

How to Run:
Clone the repository
Open in Android Studio Koala (or newer)
Sync Gradle
Run on any emulator or physical device (API 21+ is enough — you are not using Compose)
